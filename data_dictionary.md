# Data Dictionary

## Raw Data (walmart_black_friday_raw.csv)

| Column | Description |
|------|------------|
| User_ID | Unique customer identifier |
| Product_ID | Purchased product |
| Gender | Customer gender |
| Age | Age band |
| Occupation | Occupation category |
| City_Category | City tier |
| Stay_In_Current_City_Years | Tenure in city |
| Marital_Status | Marital indicator |
| Purchase | Transaction amount |

## Processed Data (customer_segments_enriched.xlsx)

| Column | Description |
|------|------------|
| user_id | Customer identifier |
| total_revenue | Total customer revenue |
| total_transactions | Number of purchases |
| avg_purchase | Avg purchase value |
| value_segment | Top 20%, Mid 30%, Bottom 50% |
